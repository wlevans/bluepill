# uart_int

This is a simple program that shows how to user interrupts with the USART. It creates (transmits) a simple menu and receives user input to controll the state of the onboard LED.

## USART 1 Configuration

|Parameter    | Value |
|:------------|------:|
|Baudrate     | 38400 |
|Data Bits    | 8     |
|Stop Bits    | 1     |
|Parity       | None  |
|Flow Control | None  |
